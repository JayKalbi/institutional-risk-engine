⚙️ System Architecture

Built for the scale and security demands of modern financial institutions.

01

Quantitative Tabular Engine

A highly-tuned LightGBM engine computes precise Probability of Default (PD) using non-linear tabular feature interactions on HMDA data.

02

Qualitative Generative Engine

A QLoRA fine-tuned Mistral-7B analyzes the loan officer's narrative and synthesizes structured memorandums using the institutional 5 C's of Credit framework.

03

Hybrid Meta-Learner

A Logistic Regression meta-model optimally fuses the probabilities of the tabular model and the LLM's classification to output a final, highly accurate default risk score.
